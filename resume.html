As a dedicated Web Penetration Tester and Cybersecurity Analyst, I aim to apply my expertise in identifying and exploiting security vulnerabilities to strengthen the digital defenses of modern organizations. With hands-on experience in conducting security assessments, exploiting web application flaws (such as XSS, SQLi, RCE), and leveraging tools like Burp Suite, Metasploit, and OWASP ZAP, my objective is to contribute proactively to threat detection and prevention. Technologies

  • Burp Suite – A powerful tool for identifying and exploiting web vulnerabilities through manual and automated testing.
  • OWASP ZAP – Open-source tool for scanning websites to detect security flaws like XSS, SQLi, etc.
  • Metasploit – An exploitation framework used to test and validate security vulnerabilities on remote systems.
  • Wireshark – A network protocol analyzer used to capture and inspect packets in real time for troubleshooting, analysis, and intrusion detection.
  • Nessus – A vulnerability scanner that identifies missing patches, misconfigurations, and known security issues across systems and networks.
  • Wifite – An automated wireless attack tool used for auditing Wi-Fi networks by capturing handshakes and testing weak passwords using various attack techniques.
  • Airmon-ng – A tool from the Aircrack-ng suite used to enable monitor mode on wireless interfaces, allowing for packet capture and wireless network analysis.
  • Nikto – Scans web servers for outdated components, dangerous files, and misconfigurations.
  • PHP/MySQL – Backend technologies for creating interactive and database-driven web apps.
  • Laravel – A PHP framework offering a clean, secure, and scalable approach to web development using MVC architecture.
